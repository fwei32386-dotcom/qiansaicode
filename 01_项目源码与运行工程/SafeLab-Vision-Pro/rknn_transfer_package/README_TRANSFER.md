# SafeLab RKNN Transfer Package

Contents:
- models/onnx/best.onnx: YOLOv8n 7-class ONNX model exported on Windows.
- models/labels.txt: class labels.
- models/calibration/images/: representative calibration images.
- calibration.txt: image list for RKNN quantization.
- test_images/: quick inference test images.
