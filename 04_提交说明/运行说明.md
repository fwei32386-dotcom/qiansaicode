# 运行说明

## 电脑端网页

```powershell
cd .\01_项目源码与运行工程\SafeLab-Vision-Pro
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python tools/serve_live_dashboard.py --host 127.0.0.1 --port 60331
```

打开：

```text
http://127.0.0.1:60331/
```

## 板卡参数

按实际板卡 IP 修改：

```powershell
$env:SAFELAB_BOARD_HOST="192.168.0.232"
$env:SAFELAB_BOARD_USER="root"
$env:SAFELAB_BOARD_PASSWORD="root"
```

## 摄像头预览

```powershell
python tools/start_board_camera_live_preview.py --host $env:SAFELAB_BOARD_HOST --username $env:SAFELAB_BOARD_USER --password $env:SAFELAB_BOARD_PASSWORD
```

## 双模型检测桥接

```powershell
python tools/run_live_dual_model_bridge.py --host $env:SAFELAB_BOARD_HOST --username $env:SAFELAB_BOARD_USER --password $env:SAFELAB_BOARD_PASSWORD
```

## 测试

```powershell
python -m unittest tests.test_rule_dsl_engine tests.test_live_dashboard tests.test_dashboard_input_source -v
```
